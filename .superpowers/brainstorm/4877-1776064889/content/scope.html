Escopo do Plano 3 — Dashboard

Quanto do dashboard implementar agora? O design-os já tem tudo prototipado.

A

Dashboard Completo

Tudo de uma vez: mapa + ranking + espécies/iscas em alta + heatmap semanal + detalhe pesqueiro + admin. Mais trabalho, mas entrega o MVP inteiro do PRD.

B

Core + Detalhe (sem Admin)

Dashboard principal (mapa, ranking, espécies, heatmap) + página de detalhe do pesqueiro. Admin fica pro Plano 4 junto com Telegram. Foco no consumo de dados.

C

Mínimo Funcional

Só o dashboard principal: mapa com scores + ranking table + espécies em alta. Sem heatmap, sem detalhe, sem admin. Entrega rápida, vê funcionando antes de investir mais.

Minha recomendação: Opção B — dá pra ver os dados completos de cada pesqueiro sem precisar do admin ainda (os pesqueiros já estão no seed). O admin é útil mas não é bloqueante pro uso diário.

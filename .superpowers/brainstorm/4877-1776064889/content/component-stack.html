Stack de Componentes UI

Como construir os componentes do dashboard?

A

shadcn/ui + Radix UI

Componentes copiados pro projeto (Button, Card, Table, Dialog, Toggle, etc). Idêntico ao design-os. Mais setup inicial, mas resultado polido e acessível. Fácil de customizar.

B

Tailwind puro — sem biblioteca de componentes

Escrever tudo na mão com Tailwind. Zero dependências extras. Mais controle total, mas mais código repetitivo. Bom pra projeto pessoal simples.

Bibliotecas de Visualização

O PRD especifica Leaflet (mapa) e Recharts (gráficos). Confirmar:

🗺️

react-leaflet + Leaflet

Mapa interativo com tiles de satélite ESRI. Markers coloridos por score. Já prototipado no design-os.

📊

Recharts

Gráfico de barras/linhas pro score 72h no detalhe do pesqueiro. Breakdown horizontal dos fatores.

✨

Lucide React (ícones)

Fish, Anchor, Waves, Ship, Navigation, etc. Usado no design-os pra ícones do sidebar e cards.

Recomendação: shadcn/ui pra componentes (já prototipado, resultado profissional) + as 3 libs de visualização (Leaflet, Recharts, Lucide). É mais setup mas é o correto pro resultado que o design-os mostra.
